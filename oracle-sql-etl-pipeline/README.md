# Cloud-Style Data Processing Pipeline using Oracle SQL

## Overview
This project simulates a real-world cloud data engineering pipeline using Oracle SQL with ETL and analytics layers.

## Architecture
Raw Data → ETL Processing → Clean Data → Analytics Layer

## Tech Stack
- Oracle SQL
- ETL Concepts
- Data Modeling
- SQL Optimization

## Key Features
- Raw data ingestion layer
- Data cleaning and transformation (ETL process)
- Business rule implementation (categorization logic)
- Aggregated analytics layer (customer insights)
- Query optimization using indexing

## Learning Outcome
Gained hands-on understanding of how cloud data pipelines work using SQL-based architecture.
